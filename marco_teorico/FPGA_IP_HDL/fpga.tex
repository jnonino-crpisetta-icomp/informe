\section{Field Programmable Gate Array}
	\label{sec:fpga}

	Una FPGA (Field Programmable Gate Array) es un circuito integrado digital formado por bloques
	lógicos los interconectados.

	Las interconexiones de una FPGA pueden ser conFigurados mediante un Lenguaje de Descripción de 
	Hardware (HDL). De esta manera una FPGA puede reproducir desde simples compuertas lógicas hasta 
	complejos sistemas dentro de un solo chip.

	Existen FPGAs que pueden ser reprogramadas lo que les permite una gran flexibilidad a la hora de
	diseñar circuitos digitales. Por otro lado, existen FPGAs que solo pueden programarse una única 
	vez, su principal ventaja su menor consumo aunque al momento de diseñar circuitos son mas complejas.

	En la \ref{fig:fpga01} se ve una FPGA dentro de una placa de desarrollo con diferentes periféricos ya 
	conectados.
	
	Las características físicas a tener en cuenta en la elección de una FPGA dependerán del proyecto que se
	desea emprender, pero, principalmente se consideran los siguientes puntos:
	\begin{itemize}
	  \item \emph{Cantidad de bloques lógicos}: determina el espacio disponible para implementar la lógica.
	  \item \emph{Tipo y tamaño de memoria}: pudiendo ser \emph{distribuida} o \emph{de bloques}, donde existe una 
	  		memoria dedicada incluida en la FPGA o se utilizan los bloques lógicos de la misma respectivamente.
	  \item \emph{Cantidad de puertos I/O}: usados como interface entre la FPGA y dispositivos externos.
	  \item \emph{Otros componentes internos}: Bloques de memoria, multiplicadores, PLL, CPU, conversores 
	  		A/D, bloques DSP, etc.
	\end{itemize}
	
	\begin{raggedright}
	Las FPGA ofrecen las siguientes ventajas:
	\end{raggedright}	
	\begin{itemize}
		\item Gran flexibilidad de producto.
		\item Posibilidad de actualizar tanto el firmware como el hardware.
		\item Reutilización del hardware.
		\item Son dispositivos reconFigurables.
		\item Bajo costo respecto a los ASIC.
		\item Ejecución de circuitos más rápida que en otros dispositivos reprogramables.
		\item \textbf{La ejecución de cada bloque se realiza concurrentemente, a diferencia de un micro controlador.}
		\item Gran utilidad para diseño y testing de prototipos. 
	\end{itemize}
	\begin{raggedright}
		Por otro lado, tienen las siguientes desventajas:
	\end{raggedright}
	\begin{itemize}
		\item Más costosas que los microprocesadores equivalentes.
		\item Mayor complejidad de diseño (soft+hard).
		\item Retardos de propagación mayores a los existentes en ASIC (Application Specific Integrated Circuit).
		\item Mayores costos en producción a gran escala.
		\item Consumo estático y dinámico moderados.
	\end{itemize}
